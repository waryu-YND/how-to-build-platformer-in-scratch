How to build platformer in Scratch

難易度：

高くジャンプする地面

ヒント

まず、コースに新しいルールを追加する を見て、好きな色（ここでは青色）を追加します。

ジャンプをするためには、プレイヤー本体の (Y速度::variables) に数字を入れます。
例えば、上向き矢印キーが押されたときには、[Y速度 v]を(15)にする::variables という風になっています。

そこで、青色に触れたときに、Y速度を20にして、ジャンプ台を作ってみましょう。

やってみよう

以下のプログラムを作ろう

青色に触れたとき、大ジャンプする

さらにヒント

青色に触れたときは、<[#0000FF]色に触れた::sensing> でできます。

答え

@greenFlag が押されたとき
ずっと
  もし<[#0000FF]色に触れた>なら
    [Y速度 v]を(20)にする
  end
end

これでジャンブ台が完成です。コースの幅が広がりました。 さらにコースを拡張してみましょう
