How to build platformer in Scratch

難易度：

プレイヤーが落ちたらやり直し

ヒント

スタート地点は、(x::variables) が0、(y座標::motion) が0のところです。

また、落ちたときに　(y座標::motion) は、-180以下になります。

やってみよう

以下のプログラムを作ろう

プレイヤーが落ちたとき、スタート地点に戻す

もっとヒント

「もし～なら」の中身は、<(y座標::motion)<(-180)::operator> です。

答え

@greenFlag が押されたとき
ずっと
  もし<(y座標::motion)\<(-180)::operator>なら
    y座標を(0)にする
    [x v]を(0)にする
  end
end

これを「プレイヤー本体」に作ることで、落ちたときにスタートにやり直しになります
